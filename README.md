# ✍️ Curso Completo de Markdown: De Cero a Profesional

¡Bienvenido al repositorio oficial del curso de Markdown! 🚀

Este proyecto está diseñado para enseñarte todo lo que necesitas saber sobre Markdown, el lenguaje de marcado ligero más utilizado en el mundo del desarrollo de software, la creación de contenidos y la documentación técnica.

Ya sea que quieras escribir mejores archivos `README.md` para tus proyectos en GitHub, tomar apuntes estructurados, o crear documentación impecable, este curso te guiará paso a paso.

## 🎯 ¿Cómo seguir este curso?

El contenido está estructurado en módulos progresivos. Te recomendamos empezar por el primero y avanzar secuencialmente, aunque siempre puedes usar este repositorio como guía de consulta rápida (cheat sheet) cuando necesites recordar alguna sintaxis.

Haz clic en el título de cada módulo para acceder a sus lecciones:

### [📦 Módulo 1: Introducción y Fundamentos](./01-introduccion-y-fundamentos/)
* Aprende qué es Markdown, por qué es el estándar de la industria y prepara las mejores herramientas para empezar a escribir.

### [🛠️ Módulo 2: Sintaxis Básica](./02-sintaxis-basica/)
* Domina el 90% de lo que usarás en tu día a día: encabezados, formato de texto, listas, enlaces, imágenes, citas, bloques de código, uso de HTML y emojis.

### [🚀 Módulo 3: Sintaxis Extendida](./03-sintaxis-extendida/)
* Ve un paso más allá con características avanzadas (GFM): organiza datos en tablas, crea listas de tareas interactivas, añade notas al pie, resalta texto clave y utiliza vistosas alertas o paneles de colores.
