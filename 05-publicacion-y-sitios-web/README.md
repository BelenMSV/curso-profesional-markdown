# Módulo 5: Publicación y Sitios Web

¡Llegó el momento de compartir tu trabajo con el mundo! 🌐

En este módulo dejamos atrás la escritura local para adentrarnos en cómo Markdown se ha convertido en el motor principal de la web moderna y la publicación de documentos. Aprenderás a transformar tus archivos `.md` en páginas web reales, documentos interactivos e informes profesionales.

## 📚 Índice de contenidos

Haz clic en cada uno de los temas para acceder a su lección correspondiente:

1. [**Generadores de Sitios Estáticos (SSG)**](./01-generadores-sitios-estaticos/) - Descubre cómo herramientas como Astro, Hugo o Docusaurus convierten carpetas llenas de Markdown en sitios web ultrarrápidos.
2. [**Introducción a MDX**](./02-introduccion-mdx/) - Aprende la evolución interactiva de Markdown que te permite incrustar componentes de React o Vue directamente en tus textos.
3. [**Exportación Universal con Pandoc**](./03-exportacion-pandoc/) - Domina "la navaja suiza" de la conversión para transformar tus apuntes en archivos PDF o Word corporativos.
4. [**Alojamiento Gratuito**](./04-alojamiento-gratuito/) - Publica tu documentación y sitios web en internet a coste cero utilizando GitHub Pages o plataformas como Vercel y Netlify.
