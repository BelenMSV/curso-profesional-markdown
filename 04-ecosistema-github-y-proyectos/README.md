# Módulo 4: Ecosistema GitHub y Proyectos

¡Bienvenido al mundo real! 🌍

Saber escribir en Markdown es solo la mitad del camino. La otra mitad es saber **dónde y cómo aplicarlo** para que tus proyectos destaquen. 

En este módulo nos centraremos en el ecosistema de GitHub (y plataformas similares como GitLab). Aprenderás a estructurar repositorios profesionales, guiar a tus colaboradores y documentar arquitecturas complejas sin salir del texto plano.

## 📚 Índice de contenidos

Haz clic en cada uno de los temas para acceder a su lección correspondiente:

1. [**El `README.md` Perfecto**](./01-el-readme-perfecto/) - Descubre la estructura ideal y cómo usar insignias (badges) para dar una imagen profesional.
2. [**Plantillas de GitHub**](./02-plantillas-de-github/) - Estandariza la forma en que otros contribuyen a tu proyecto creando plantillas automáticas para *Issues* y *Pull Requests*.
3. [**Metadatos (YAML Frontmatter)**](./03-metadatos-frontmatter/) - Aprende a definir variables ocultas (como títulos, autores o fechas), fundamentales para blogs y wikis.
4. [**Diagramas con Mermaid.js**](./04-diagramas-mermaid/) - Dibuja espectaculares diagramas de flujo y gráficos circulares escribiendo únicamente bloques de código.
