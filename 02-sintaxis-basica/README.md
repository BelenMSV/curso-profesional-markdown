# Módulo 2: Sintaxis Básica

¡Entramos en materia! 🛠️

En este módulo vamos a ensuciarnos las manos y a aprender la sintaxis principal de Markdown. Al finalizar estas lecciones, serás capaz de estructurar, dar formato y enriquecer cualquier documento de texto o archivo `README.md` para tus proyectos.

Hemos dividido la sintaxis en pequeñas píldoras para que sea muy fácil de asimilar. 

## 📚 Índice de contenidos

Haz clic en cada uno de los temas para acceder a su lección correspondiente:

1. [**Encabezados y Párrafos**](./01-encabezados-y-parrafos/) - Aprende a estructurar tu documento y a separar ideas correctamente.
2. [**Énfasis y Reglas Horizontales**](./02-enfasis-y-reglas/) - Destaca texto con negritas, cursivas y divide secciones con líneas visuales.
3. [**Listas Ordenadas y Desordenadas**](./03-listas-ordenadas-y-desordenadas/) - Organiza la información mediante viñetas, enumeraciones y sub-listas.
4. [**Enlaces e Imágenes**](./04-enlaces-e-imagenes/) - Conecta tu documento con internet y hazlo visualmente atractivo.
5. [**Citas y Bloques de Código**](./05-citas-y-bloques-de-codigo/) - Destaca frases importantes y muestra código fuente con resaltado de sintaxis profesional.
6. [**HTML, Emojis y Escapado**](./06-html-emojis-y-escapado/) - Dale un toque personal con iconos, evita errores de formato y descubre el poder del HTML integrado.
